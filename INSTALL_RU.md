# Установка Chrome Video Translator

1. Скачайте файл `chrome-video-translator.zip` из этого репозитория.
2. Распакуйте архив в любую удобную папку.
3. Откройте Chrome и перейдите на страницу `chrome://extensions`.
4. Включите переключатель `Режим разработчика` в правом верхнем углу.
5. Нажмите `Загрузить распакованное расширение`.
6. Выберите папку `chrome-video-translator`, которая появилась после распаковки архива.
7. Откройте страницу с видео, включите субтитры и настройте язык/голос через popup расширения.

Рекомендации:
- Если Chrome уже был открыт на тяжёлой странице, после установки один раз обновите вкладку.
- Для более мягкого звучания начните с `Speech rate 0.90-0.94` и `Speech pitch 0.94-0.98`.
- Если сайт очень тяжёлый, сначала включите субтитры на видео, а потом открывайте popup расширения.
