# Chrome Video Translator

Готовая сборка Chrome-расширения для перевода видео-субтитров с озвучкой перевода голосом.

Что внутри:
- `chrome-video-translator.zip` — готовый архив для установки
- `INSTALL_RU.md` — инструкция по установке на русском

Как установить:
1. Скачайте `chrome-video-translator.zip`.
2. Распакуйте архив.
3. Откройте в Chrome страницу `chrome://extensions`.
4. Включите `Режим разработчика`.
5. Нажмите `Загрузить распакованное расширение`.
6. Выберите папку `chrome-video-translator` после распаковки.

Прямая ссылка на архив:
- https://github.com/sergeylkn/NewRepo/raw/master/chrome-video-translator.zip
